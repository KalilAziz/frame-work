Parâmetros da função

Os parâmetros de uma função tem a função de receber os valores dos argumentos, onde poderemos utilizar esses valores para fazer alguma operação na função ou qualquer coisa do tipo. Os parametros serão utilizados em ordem da esquerda para a direita, ou seja, quando o argumento mandar um valor, o primeiro parâmetro a receber um valor será o primeiro da esquerda, e por ai vai. Diferentemente de outras linguagens, o js não liga se vc usou o parâmtro criado ou enviou mais argumentos que o necessário, mas o recomendado é enviar apenas os argumentos que serão usados.

Parâmetros como variáveis

O motor do JavaScript trata os parâmetros como variáveis que poderão armazenar valores que serão usados ao decorrer da função. Caso coloquemos um parâmetro que não receba um valor de um argumentos, terá seu valor como undefined, assim como uma variável que não recebeu valor.

Tratamento de valores recebidos

O motor do JavaScript trata os parâmetros como variáveis que poderão armazenar valores, e caso não receba nenhum valor do argumento, terá seu valor como undefined, assim como uma variável que não recebeu valor. Caso estejamos realizando alguma operação matemática, o undefined fará com que a operação trave, retornando um NaN. Para que possamos prevenir isso, podemos setar um valor padrão aos parâmetros, para que se ela não receba valores dos argumentos, esses valores setados irão substituir e evitar que aconteça uma falha no sistema. Para que possamos fazer esse tratamento, precisamos apenas atribuir o valor desejado no parâmetro desejado.

Pulando argumentos para usar nos parâmtros

O motor do JavaScript atribui automaticamente os valores dos argumentos em ordem com os parâmetros, caso desejamos pular um parâmetro, não será possivel faze-lo. Para que possamos pular, teremos que atibuir ao parâmetro um valor undefined por meio do argumento. Fazendo isso, o valor que a função irá usar será o valor padrão que definimos com o tratamento da função.

atribuição via desestruturação com parâmetros - Objetos

Podemos mandar qualquer tipo de dado por meio dos argumentos, para que possamos utilizar na função, incluindo objetos. Mas para conseguirmos usar os dados brutos enviados de um objeto inteiro enviado pelos argumentos nos parâmetros, será necessário fazer uma atribuição via desestruturação. Mas é bem simples e precisamos apenas abrir e fechar chavez nos parâmetros da função, onde colocaremos os nomes das chavez que queremos usar dos objetos. Posteriormente, basta usar normalmente os valores na função. Vale lembrar que podemos enviar um objeto inteito nos argumentos da função, ou podemos atribuir esse objeto a uma variável e enviar apenas o nome da variável.
