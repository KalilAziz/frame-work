Funções recursivas

As funções recursivas é quando uma função se chama ela mesma de volta, fazendo algo muito parecido que as estruturas de repetição fazem, mas temos que ter cuidado com o limite de recursividade, pois temos um limite máximo definido pelo motor do js, travando a recursividade. Para que possamos utilziar uma função recursiva, teremos que usar um If, com uma condição, onde se for verdadeira, irá executar o return. No final da função irá ter a chamada da pro[ria função. Não podemos esquecer do incrementador, pois se não, a função ficará em looping infinito.

A teoría por trás será simples, uma vez que a função for iniciada, ela será percorrida, como o return para encerrar a função está dentro do if, e no final da função no lugar do return vai estar a chamada da própria função, a função só será finalizada se a condição do if for satisfeita e o return for lido. E para que essa condição seja satisfeita, iremos chegar nela com base no incrementador ou decrementador.
