Função de callback

O problema do desenvolvimento web, é que precisamos conectar na base de dados, buscar recursos na base de dados, executar alguma coisa na tela, buscar recursos em uma API, solicitar carregar um arquivo JSON... ou seja, na web, precisamos fazer coisas que não temos como controlar quanto tempo elas irão demorar para fazer esse processo, e se surgir uma situação onde precisamos executar as funções em ordem, o sistema irá ficar descontrolado e provavelmente acontecerá alguma falha. Para que possamos controlar a ordem de execução, iremos utilizar o callback, que nos possibilitará fazer esse controle na ordem de execução.
