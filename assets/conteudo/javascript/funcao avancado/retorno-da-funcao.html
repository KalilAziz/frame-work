Retorno da função

O retorn de uma função tem o papel de encerrar uma função e retornar um valor quando a função for chamada.

Funções sem retorn

O retorn de uma função tem o papel de encerrar uma função e retornar um valor quando a função for chamada, mas não é totalmente necessário em uma function, onde pode existir funções sem retorn. Mas a diferença é que quando essa função for chamada, não irá retornar um valor, mas apenas executar uma ação que estiver dentro do bloco de notas, seja para exibir alguma coisa para o usuário ou alguma modificação na estrutura do programa.

Return com objetos

Com a utilização do retorn, podemos retornar qualquer tipo de valor, mesmo que seja estruturas completas de dados, como objetos. Para que possamos retornar objetos, precisamos criar um objeto com seus atributos com os mesmos nomes dos parâmetros da função, dessa forma, não será necessário criar os atributos e atribuir o valor manualmente, automatizando esse processo. Para conseguir acessar os valores dos atributos do objeto, a melhor forma é atribuir a chamada da função com os argumentos a uma variável, e com o nome dessa variável, iremos usar a marcação de ponto e o nome do atributo que queremos.

Chamada de uma função no retorn outra função

Podemos também retornar a chamada de outra função, onde teremos que criar uma função dentro da outra. Vale lembrar que como os parâmetros são tratados como variáveis, os parâmetros da função externa poderá ser acessado pela função interna.
