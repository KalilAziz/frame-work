Template String

Template String é uma forma que utilizamos, onde nos permite ter expressões embutidas. Uma de suas grandes utilizações é quando iremos exibir um resultado final no console.log(), onde normalmente utilizávamos aspas, poderemos usar template string. Para conseguirmos utilizar, basta em vez de colocar aspas, colocarmos crazes. A diferença do template string com as aspas é que ela não dá conflito se tiver alguma palavra no meio que precise utilizar aspas, e podemos controlar exatamente o que iramos exibir para o usuário. A utilização de variáveis em template string é um pouco especial, onde temos que colocar ela dentro de uma expressão, sendo ela: ${}, onde basta colocar as variáveis dentro das chaves que elas poderão ser usadas normalmente.
