innerHTML

O innerHTML tem a capacidade de escrever em elementos desejados. Para que possamos utilizar o innerHTML, temos que colocar o nome no elemento e posteriormente .innerHTML, depois somente colocar um sinal de atribuição e colocamos o que precisamos, seja uma variável, tags html ou seja um texto, lembrando que se formos colocar caracteres, tem que ser dentro de template string.

OBS: O problema é que se já existia conteudo anteriormente dentro do elemento, ele sempre apagará todo conteúdo e colocará o que atribuimos ao innerHTML. Para resolvermos isso, além de colocar o operador de atribuição, colocamos o de atribuição com o "+", que irá pegar o conteudo que tinha anteriormente e adicionar o conteúdo que desejamos.

Para demonstramos o funcionamento do innerHTML, iremos utilizar o elemento body do html. E para isso, temos que selecioná-lo. Para selecionar o body, temos que colocar document.body, onde document significa todo o corpo em si do site.
