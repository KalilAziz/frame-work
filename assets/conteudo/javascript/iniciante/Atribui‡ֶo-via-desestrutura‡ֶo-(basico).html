Atribuição via desestruturação (basico)

A atribuição via desestruturação serve para que possamos atribuir valores a variáveis de maneira bem mais prática e poderemos utilizar para reatibuir os valores das variáveis entre si mesmo. E para que possamos utilizar atribuição via desestruturação, temos que colocar dois pares de colchetes, que serão separados por um sinal de =, onde do lado esquerdo determinamos as variáveis iniciais e do lado direito colocamos as variáveis na ordem que queremos que os valores sejam atribuidos.
