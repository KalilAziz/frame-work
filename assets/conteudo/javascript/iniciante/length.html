Length

O lengt é muito utilizado para que possamos determinar a quantidade de valores ou o tamanho de uma estrutura de dados, onde por exemplo, podemos verificar a quantidade de caracteres de uma String ou o tamanho de uma array. Para utilizar o length, basta apenas colocarmos o nome da variável que carrega a estrutura de dados e adicionar um .length.
