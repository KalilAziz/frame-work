Conversão String em Number

A converção de String em Number é muito importante, porque iremos deparar em vários cenários na construção de um sistema que obteremos valores numéricos do banco de dados ou de algum lugar em string, que teremos que fazer algum tipo de operação, e para isso, eles terão que ser do tipo Number. Para que possamos fazer essa converção, iremos utilizar o Number(), que normalmente será atribuido a alguma variável, e dentro dos parenteses temos que colocar o nome da variável que carrega o valor em string que desejamos converter.
