getElementById

Podemos utilizar o getElementById para selecionar os elementos htmls, e para isso, temos que colocar um id em algum elemento na página HTML, posteriromente utilizamos o document.getElementById(''), que será atribuido a uma variável, onde entre parenteses, colocamos o id selecionado, lembrando que tem que ser colocado entre aspas. Temos que colocar o document pois ele representa todos a estrutura que contem os códigos HTML, então o getElementById('') vai procurar elemento que possue uma ID específica que está dentro da estrutura do HTML.
