Atribuições

A atribuição é a maneira de colocar operadores aritméticos antes do sinal de =

Onde um exemplo, se temos a seguinte expressão: elemento1 += elemento2, é o mesmo que atribuir ao elemento1 a soma dele mesmo com o elemento2
