Aspas e barras em uma String

Para poder colocar aspas duplas dentro da frase, temos que iniciar a string com aspas simples, e se quisermos colocar palavras com aspas simples no meio da frase, temos que começar com aspas duplas.

Para que possamos ter duas aspas iguais, temos que colocar essa barra invertida antes das novas aspas para que não dê conflito, gerando o mesmo resultado.

Se quisermos colocar uma barra invertida no meio do texto, precisamos colocar duas barras.

Indexação de String

Strings tem as características de ser indexadas, ou seja, cada caracter tem sua posição, começando do 0. Então podemos, ao exibir uma String, colocar a posição entre colchetes para que possamos obter aquele caractere específico da posição. Caso coloque em uma posição que não exista a parte da string, mostrará undefined

Concatenação de uma String

Para utilizar o .concat(), precisamos primeiramente colocar o nome da variável da string que desejamos concatenar, depois colocamos o .concat(), e dentro dos parenteses, colocaremos o que queremos concatenar entre aspas

Podemos utilizar o sinal aritmético de + para fazer concatenação, onde precisamos apenas colocar o sinal entre duas Strings. Caso a String esteja armazenado em uma variável, não faz diferença

Com a utilização das crazes, estamos utilizando template string, onde tudo dentro das crazes será exatamente o conteúdo exibido, ou seja, se colocarmos duas strings juntas sem espaços, elas serão exibidas juntas. Para a utilização de variável, assim como o padrão de template string, tem que ser colocado dentro de ${}.

Substituição em uma String

Podemos utilizar o .replace() para substituir uma palavra por outra. E para que possamos fazer essa substituição, precisamos primeiro colocar o nome da variável que carrega a string, postriormente o .replace(). Dentro dos parenteses, vamos passar dois valores, sendo um deles a palavra ou caracteres que sesejamos substituir, e o outro o que vamos colocar no lugar, os valores tem que estar separados por vírgula e entre aspas.

OBS: Vale lembrar que tudo em uma String são caracteres, seja espaços, traços, qualquer coisa
