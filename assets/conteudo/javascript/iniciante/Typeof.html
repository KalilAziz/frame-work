Typeof

O Typeof é utilizado para que possamos identificar o tipo de alguma estrutura de dados, podendo ser: String, number, object e Boolean. Para que possamos utilizá-lo, primeiro temos que colocar typeof e o nome da variável que carrega o valor, tendo os dois nomes separados.
