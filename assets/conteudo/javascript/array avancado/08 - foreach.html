forEach

O forEach é uma variável do for muito interessante onde podemos obter certos tipos de valores, seu diferencial é que podemos obter automaticamente, sem precisar colocar condições, variáveis de start ou incrementador. Mas o forEach tem uma limitação, conseguindo trabalhar apenas com arrays, onde ele percorre o array automaticamente e nos fornece três valores, que são: o valor da posição que o forEach está, a posição que o valor está no array e o proprio array. Para que possamos usar o forEach, temos que colocar o nome do array, marcação de ponto, e depois temos que colocar forEach(). Dentro dos parenteses, temos que colocar uma função anônima (podendo ser arrow function), onde podemos escrever o nome de três parâmetros, sendo que o primeiro deles significará o valor, o segundo significará a posição do valor no array e o terceiro será o próprio array. Posteriormente, basta usar esses paramentos quer carrega os valores do jeito que quiser no corpo da função.

Semelhanças com filter, map e reduxe

A forma de iterar com forEach é muito parecido com map e filter, onde precisamos passar uma function dentro dos parenteses, onde nos parâmetros podemos recever três valores vindos do array. Podemos com o forEach replicar todos os tipos de for, e inclusive o reduce.
