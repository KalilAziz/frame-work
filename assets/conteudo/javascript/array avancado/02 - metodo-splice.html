Método splice

O método Splice faz o que o pop, push, shift e unshift faz e mais um pouco. No splice() podemos passar vários argumentos, sendo que o primeiro é qual indice queremos mecher, depois colocamos quantos elementos queremos remover, e por fim colocamos os elementos que queremos adicionar. Exemplo:

nomes.splice(indice, delete, elem1, elem2, elem3)

Simular a função pop: So colocar para remover um elemento na posição -1: splice(-1,1). Assim como o pop, ela tambpem retorna um elemento, só que é em formato de array, por que podemos remover mais de um elemento.

Simular a função shift: So colocar para remover um elemento na posição 0: splice(0,1)

Simular a função push: So colocar para mecher no elemento .length(que será a ultima posição vazia) e adicionar o elemento sem remover nenhum elemento: splice(nome.length, 0, elem)

Simular a função unshift: So colocar para mecher no elemento 0, sem remover nd e colocar os elementos que queremos adicionar: splice(0,0, "elem1")
