Reduce

O reduxe é uma função específica do array a qual podemos reduzir o array em um único elemento. Para que possamos usar o reduxe, temos que declarar uma variável e atribuir a ela o nome do nosso array juntamente com marcação de ponto e reduxe(). Dentro dos parenteses, temos que passar duas informações, sendo uma delas a função, podendo ser o nome da função e declará-la em outro lugar do sistema, ou podemos apenas criar uma função anônima dentro do próprio parenteses, e o segundo valor é o número start do nosso parâmtreo de acumulador. Uma das vantagens de utilizar o reduxe é que, como parâmetro da função, podemos colocar quatro nomes, que automaticamente irão carregar dados do nosso array principal, sendo eles o acumulador, que pode ser usado para guardar algum valor durante a execução da função, valor do array, o indice do valor e o array completo, onde o reduxe irá entrar em looping de repetição até que todos os valores sejam lidos. No retorn da função é onde o novo array será criado e acumulado, quando todos os valores do array tiverem sido lidos, o retorn irá retornar o novo array como resultado.

Obs: Um exemplo de utilização do reduxe é: se temos uma lista de números em um array, podemos usar o reduce para somar o valor total do array. Podemos utilizar o reduce para fazer o filter e map, mas não é utilizado e nem ideal. Diferentemente do filter e map, no reduce o primeiro atributo que recebemos não é o valor do elemento, mas o acumulador, depois recebemos o valor, indice e array. O acumulador é uma variável específica criada para colocar as somas dos valores. Outra diferença é que o método reduce recebe dois valores, sendo a função de callback e o valor inicial do acumulador.
