Arrays

Um array é uma lista, uma coleção de coisas com divisões e podemos acessar cada valor separadamente. Para criar uma array, utilizamos uma variável, um nome, um sinal de igual e colchetes, onde dentro do colchetes colocamos oque quisermos separados por vírgulas, onde são indexadas, começando por 0.

Index de uma array

Em geral, sempre quando tempo algum tipo de estrutura de dados em JavaScript, normalmente funciona sempre da mesma forma, começando sempre a partir de 0 e com array não é diferente. Seus valores são separados por virgulas, se for Strings, seus valores estarão entre aspas, caso contrário, qualquer outro tipo de valor, não será necessário. E como já joi dito, o index começa com 0

Para fazer a exibição de uma array, precisamos apenas colocar o nome da variável com a qual ela foi criada

Se quisermos exibir um valor específico do array, é necessário especificar a posição com o qual esse elemento representa dentro do array, e para isso, colocamos o nome do array, colchetes e dentro deles, colocamos sua o valor do index que ele representa na estrutura

Caso tentamos exibir o valor de uma posição que não existe, o motor do JS não identificará como inválido NaN, mas como valor undefined

Remoção de valores (espaço vazio) de uma array

Para que posssamos deletar um valor do array, podemos utilizar o delete. Para que possamos usá-lo, precisamos colocar o delete, e posteriormente referenciar o valor do array que desejamos fazer a remoção, com o nome do array e a posição do valor pelo index dentro de colchetes.

A diferença do delete para os modos de remoção é que nessa forma, a posição dos outros elementos não serão afetados, porque o valor não vai ser retirado verdadeiramente do array, pois sua posição permanecerá lá, mas seu valor será vazio.

Esse é um jeito interessante de remover um elemento, caso não queiramos que todos os valores posteriores não tenham suas posições reatribuidas.

Inserção de valores na ultima posição(.length) de uma array

Para adicionar um valor na ultima posição de uma array, podemos apenas colocar da forma que foi mostrada, com a diferença que escolheríamos a posição vazia para inserir o valor, mas temos um problema, que é que teríamos que saber o tamanho da array, para saber o ultimo indice e colocar na próxima posição, sendo a vazia. Para tentar passar por cima desse problema é na utilização do .legth.

O .length tem a capacidade de exibir a quantidade de elementos que determinada estrutura de dados possue. O .length é perfeito para essas situações de inserção, por exemplo: em uma array, com o qual o ultimo valor tem seu index como 4, e como a indexação começa por 0, esse array tem 5 elementos, e o número 5 é exatamente o valor que o .length mostra. Então, como dá pra perceber, o .length sempre mostra um a mais que o index atual, e é exatamente o que precisamos, inserir o valor uma posição a mais que o ultimo valor do array.

Para utilizar o .length nessa situação, basta apenas colocar o nome do array e colchetes, exatamente como visto anteriormente, mas em vez de colocar um número dentro dos colchetes, especificando a posição que o valor será inserido, colocamos alunos.length, que gerará esse número automaticamente, posteriormente, basta colocar o sinal de atribuição e o valor que será atribuido aquela posição do array

Remoção de valores na primeira posição de uma array

Se quisermos remover um elemento no início do array, a lógica é mais complicado, pois ao remover um elemento no início, temos que mover todos os outros valores do array para ocupar o lugar vago, então o index de todos os valores serão alterados. Então se tivermos alguma parte do programa que tem o papel de ler os valores do array e dependa da posição deles, temos que tomar cuidado, pois podemos causar problema em todo o código.

Para conseguirmos fazer a remoção do primeiro elemento do array, precisamos utilizar o .shift(). Para que possamos usar o .shift(), primeiramente temos que colocar o nome do array que queremos fazer a remoção e posteriormente .shift().

O diferencial é que podemos colocar essa remoção atribuido a uma variável, onde podemos recuperar esse valor que foi removido caso desejamos utilizar essa variável posteriormente no sistema.
