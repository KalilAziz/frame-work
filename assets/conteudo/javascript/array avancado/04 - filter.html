Filter

O filter é uma função específica do array a qual podemos fazer um filtro de um array, onde sempre temos como retorno um array com a mesma contidade de elementos ou menos que o array original. Para que possamos usar o filter, temos que declarar uma variável e atribuir a ela o nome do nosso array juntamente com marcação de ponto e filter(). Dentro dos parenteses, teremos que colocar uma função, podendo ser o nome da função e declará-la em outro lugar do sistema, ou podemos apenas criar uma função anônima dentro do próprio parenteses. Uma das vantagens de utilizar o filter é que, como parâmetro da função, podemos colocar três nomes, que automaticamente irão carregar dados do nosso array principal, sendo eles o valor, o indice do valor e o array completo, onde o filter irá entrar em looping de repetição até que todos os valores sejam lidos. No retorn da função é onde o novo array será criado e acumulado, quando todos os valores do array tiverem sido lidos, o retorn irá retornar o novo array como resultado.
