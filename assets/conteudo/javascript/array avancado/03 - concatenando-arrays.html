Concatenação com Arrays

Para que possamos fazer a concatenação de arrays, não podemos fazer de jeito nenhum utilizando o sinal de + como fazemos com string, onde com arrays, ficará todo bagunçado e travará o sistema. Para que possamos fazer da forma correta, podemos fazer de duas maneiras, sendo com a função concat ou com o rest operator, se tornando um spread.

concat()

Podemos usar a função concat para que possamos concatenar arrays. Onde teremos que declarar uma variável, onde teremos que colocar o nome da array principal, posteriormente iremos colocar marcação de ponto, e a função concat(). Dentro dos parenteses iremos colcoar o array que desejamos concatenar dentro do array principal. Mas podemos colocar não apenas outra array, mas um conjunto de valores, como [4, 5, 6, 7]. Podemos mandar mais de uma concatenação, mas precisamos separar por vírgula. Lembrando que esse método, concatenamos tudo, não apenas arrays, podemos colocar valores de string também para ser concatenados.

Spread

//Podemos usar o rest operator também, mas ele se tornará um spread, que vai espalhar o array, se tornando outra forma de concatenação, se tornando uma forma mais fácil de concatenar, funcionando em objetos também, não apenas em array. Para que possamos utilizálo, temos que declarar uma variável, onde iremos atribuir um abre e fecha colchetes, igual a como declaramos uma array. Posteriromente, iremos colocar as arrays que desejamos concatenar em uma única array, mas temos que colcoar ... antes dos nomes das arrays concatenas. Mas podemos colocar não apenas outra array, mas um conjunto de valores, como [4, 5, 6, 7]. Podemos mandar mais de uma concatenação, mas precisamos separar por vírgula. Lembrando que esse método, concatenamos tudo, não apenas arrays, podemos colocar valores de string também para ser concatenados.
