querySelectorAll

Para que possamos selecionar um grupo elementos, podemos utilizar o querySelectorAll, que tem a capacidade de selecionar elementos como selecionaríamos classes ou ids, funcionando muito parecido com o QuerySelector, mas o QuerySelectorAll tem a capacidade de selecionar todos um grupo do mesmo tipo de elemento, enquanto o QuerySelector seleciona o primeiro elemento especificado. Se for selecionar um elemento html, colocamos apenas o seu nome, se for selecionar utilizando classes, temos que colocar um '.' e o nome da classe e se fomos selecioanr um id, colocamos '#' e o nome do id, lembrando que temos que colocar entre aspas, mas podemos colocar entre crases, ou aspas duplas.

Obs: Se quisermos pesquisar em todo o documento, podemos colocar document.querySelectorAll, com o document. Mas se quisermos fazer uma busca mais específica, como apenas dentro de uma classe, selecionamos ela primeiro com querySelector e depois usamos o querySelectorAll.

Obs2 : Para que possamos colocar formatações nessas seleções, temos que fazer dentro de um For of.
