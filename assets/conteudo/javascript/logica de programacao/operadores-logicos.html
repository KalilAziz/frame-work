Operadores lógicos

Os operadores lógicos servem para checar mais de uma comparação, sendo eles: &&, ||, !                                                                                                                                                               

&& -> AND -> E: onde eles terão que comparar as duas condições para que seja dado como verdadeiro, caso uma delas seja, falsas, tudo será dado como falso

|| -> OR -> OU: Apenas uma das condições tem que ser verdadeira, não importa qual, será dado como verdadeiro, caso contrário, será dado como falso

! -> NOT -> NÃO: Sinal de negação: Ele inverte o valor de uma operação

Operadores lógicos: &&

O Operadore lógico && serve para comparar as duas condições para que seja dado como verdadeiro ou falso, onde as suas precisam ser verdadeiras para que seja retornado true, caso uma delas seja, falsas, será dado como false

Operadores lógicos: ||

O Operadore lógico || serve para comparar as duas condições para que seja dado como verdadeiro ou falso, onde pelo menos uma das condições tem que ser verdadeira para que seja retornado true, e para que seja retornado false, as duas tem que ser falso.

See the Pen 038.2 operadores lógicos by KalilAziz (@kalilaziz) on CodePen.

Operadores lógicos: !

O Operadore lógico ! serve para inverter o resultado true ou false de um valor. Podemos acumular esse operador também, ou seja, se temos um valor true e colocar !true, ficará falso, mas se colocar !!true, ele permanecerá true, já que foi convertido para false e convertido para true novamente.
