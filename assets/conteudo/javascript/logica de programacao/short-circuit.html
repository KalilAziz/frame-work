Short Circuit

O Short Circuit faz utilização dos operados lógicos sendo eles: &&, ||, !                                                                                                                                                                                                 

&& -> AND -> E: onde eles terão que comparar as duas condições para que seja dado como verdadeiro, caso uma delas seja, falsas, tudo será dado como falso

|| -> OR -> OU: Apenas uma das condições tem que ser verdadeira, não importa qual, será dado como verdadeiro, caso contrário, será dado como falso

! -> NOT -> NÃO: Sinal de negação: Ele inverte o valor de uma operação

&&

Na execução de um código que possue um &&, onde todos precisam ser verdadeiros para que retorne verdadeiro, eles são executados em ordem, onde se encontrar um falso no meio do caminho, ele retorna o falso, se todos foram verdadeiros, ele retorna o ultimo verdadeiro.

Exemplo de utilizaçào com &&

Utilizando essa teoria para que podemos utilizar em um código mais perfomático e menos verboso, podemos fazer uma função com o dever de retornar apenas 'Oi', mas o diferencial é que podemos colocar uma variável que pode controlar o fluxo, definindo se essa função será executada ou não. Que em vez de fazer uma condição de checagem podemos usar o operador &&.

||

Na execução de um código que possue um ||, onde apenas um precisa ser verdadeiros para que retorne verdadeiro, eles são executados em ordem, onde se encontrar um verdadeiro no meio do caminho, ele retorna o verdadeiro, se todos forem falso, ele retorna o ultimo falso.
