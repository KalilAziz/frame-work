getComputedStyle

O getComputedStyle tem a capacidade de extrair todos os tipos de informações css aplicados a um elemento específico. Onde posteriormente precisamos apenas escolher que tipo estilo queremos usar, lembrando que para estilizações css que possuem dois nomes, como background color, precisamos formatar em CamelCase, ou seja, não pode ter espaços no nome, e se tiver uma segunda palavra, a primeira palavra começa com letra minuiscula e a segunda palavra com letra maiúscula. Então, background color vira backgroundColor. Para que possamos selecionar o elemento que desejamos ler todas as características css, temos que primeiro criar uma variável com o nome e atribuir getComputedStyle() a ela, onde dentro dos parenteses colocamos o elemento que queremos ler. Para que possamos selecionar qual css gostaríamos que visualizar, basta criar outra variável com o nome, onde iremos atribuir o nome da primeira variável com marcação de ponto, onde iremos colocar o nome do css.
