keyCode

Podemos usar o keyCode para identificar qual tecla está sendo clicado, onde podemos usar juntamente com um if e executar alguma ação no sistema. Para que possamos usar o keyCode, primeiro precisamos colocar um addEventListener onde quer que queiramos que o evento aconteça, onde iremos passar como evento "keypress". Iremos colocar um parâmetro dentro da função que iremos usar com marcação de ponto juntamente com o KeyCode dentro dos parenteses do if. Depois de colocar o parâmtro, marcação de ponto e KeyCode, precisamos usar o sinal de igualdade estrita e colocar o valor da tecla do tecla, no caso do enter, o valor é 13. Para que possamos saber qual é o valor de cada tecla do teclado, iremos dar um console.log no parâmetro da função, onde pelo navegador, iremos conseguir ver no console o valor da tecla.
