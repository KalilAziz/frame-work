For in

O for in é uma variável do for, sendo muito utilizada pois nos economiza bastante tempo e código. Uma das grandes antagens do for in, é que podemos pegar diretamente os valores da estrutura de dados, podendo ser array ou objetos em um laço de repetição, onde poderemos usar da forma que queremos no sistema. Pra que possamos usar o For in, precisamos apenas declarar o for(){}, onde dentro dos parenteses, colocamos uma variável sem atribuição de valor, onde que o valor será atribuido durante a repetição e poderemos utilizar no código. Posteriormente colocamos 'in'e por fim, o nome do array ou objeto. Dentro das chavez colocamos o código que queremos executar em no laço de repetição.

For in com objetos

Para conseguirmos usar esse conceito com objetos é um pouco diferente, pois se tentarmos usar apenas o i, ele nos passará o nome dos atributos do objeto, o que pode ser o que queremos no código, mas se quisermos usar os valores dos atributos, temos que colocar o nome do objeto, e entre parenteses, colocamos o i.

OBS: Uma nova forma de chamar os valores dos atributos de um objeto é usar o nome do objeto, juntamente com o nome do atributo que queremos entre aspas e dentro de colchetes, como: objeto['atributo'].
