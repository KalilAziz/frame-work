Atribuições via desestruturações - Objetos

A atribuioção via desestruturação é uma nova forma que poderemos fazer atribuições de estruturas, podemos fazer essas atribuições de várias variáveis simuntâneamente. Com a atribuição via desestruturação, podemos "arrancar" coisas do objeto e já utilizar nessa sintax. Lembrando que se inicializar com const, todas as astribuições seguintes terão que ser com const, e com let a mesma coisa. Para que possamos utilizar atribuição via desestrituração com objetos, temos que declarar uma constante com chaves e atribuimos o objeto que queremos retirar os valores. Dentro das chaves, colocamos o nomes dos atributos que desejamos pegar dento do objeto.

Esse método facilita muito, já que anteriormente teríamos que usar o nome das chaves do objeto, com marcação de ponto, e agora podemos pegar esse valor e já criando uma variável com esse nome.

Tratamento de erros

Alguns dos erros mais comuns é que quando estamos fazendo uma atribuição via desestruturação, como temos que colocar o nome dos atributos, pode ocorrer de colocar um nome de um atributo que ainda não exista no objeto, seja por que o atributo ainda não foi inserido no objeto ou seja por conta de um erro do próprio programador. Para que possamos contornar esse problema, temos que, ao colocar o nome do atributo, atribuimos um valor fixo, para que caso o atributo não exista no objeto, esse valor será substituido pelo valor fizado.

Alteração de nomes de variáveis

Como em Atribuição via desestruturação utilizando objetos, temos que colocar o nome dos atributos para que possamos usar os valores, ficamos muito limitado a nomeclatura, sendo que se colocar um nome diferente, não iremos conseguir pegar os valores dos objetos. Para que possamos utilizar outro nome e manter a funcionalidade padrão, precisamos renomear na hora da tribuição, e para que possamos fazer isso, iremos utilizar o sinal de dois pontos ":". Para que possamos utilizá-lo, dentro das chaves, quando tivermos fazendo a atribuição via desestruturação, temos que colocar o nome do atributo que desejamos pegar seu valor, posteriormente temos que colocar o sinal de dois pontos e o novo nome que usaremos. Para que possa ficar mais completo ainda, podemos usar o método de tratamento de erro, protegendo os valores, podemos deixar apenas atribuido um espaço vazio, para que se faltar o valor, pelo menos não apareça um undefined.

Objetos dentro de objetos

Diversamente nos depareamos com arrays dentro de outros arrays, e com objetos não será diferente, para que possamos acessar os valores desse tipo de objeto, teremos que usar um método diferente. Para que conseguirmos acessar esses valores, na hora de fazer a atribuição via desestruturação, primeiro precisamos identificar o objeto que desejamos colher os valores, onde teremos que colcoar seu nome e dois pontos, posteriormente teremos que abri e fechar chavez. Dentro dessas chavez, iremos colocar, separado por virgulas, os atributos desse objeto que desejamos ter os valores.

Resto

Assim como com atribuição via desestruturação com arrays, podemos pegar o resto de um objeto, mas com uma diferença, como em arrays, temos que colocar os nomes dentro dos colchetes na ordem que queremos pegar os valores, e o resto, serão os valores posteriormes a posição que o resto foi colocado, em objetos, vai ser literalmente o resto, ou seja, todos os outros valors que ainda não tenha sido chamados, ou seja, se o primeiro nome que colocamos para fazer essa atribuição seja um atributo que esteja no meio do objeto, e o segundo nome foi o resto, o resto pegará todos os valores, do início ao fim, com exeção do valor pegado anteriormente. Para que possamos usar o resto, temos que usar o spred operator, tendo que colocar três pontos (...) e um nome, esse nome carregará o resto do array, normalmente esse nome se chama resto mesmo, ou rest.
