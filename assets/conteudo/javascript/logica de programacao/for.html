For

O for é uma das principais estruturas de repetições, estando em muitos tipos de linguagens. O for é usado para quando temos uma parte no código que precisamos repetir uma quantidade conhecida de vezes, sendo muito importante esse conceito, já que existe outras estruturas de repetições, mas seus propósitos são diferentes. Um exemplo de utilização de for: Caso seja necessário que exibimos em console.log a seguinte frase: linha 0, linha 1, linha 2... linha 5. Pode parecer simples, e fácil, por ser apenas 6 linhas de código, mas e se precisarmos fazer essa repetição com 10000 linhas de código?! E é ai que o For entra, um laço de repetição de um número connhecido de vezes que irá se repetir.

Para que possamos criar o For, primeiro precisamos declarar o For com um abre e fecha parenteses e abre e fecha chaves. Dentro dos parenteses iremos passar três informações que irão ser separado por ponto virgula (;), sendo o primeiro deles vamos criar uma variável com let, onde iremos atribuir um valor que irá startar a repetição, a segunda informação é a condição de parada, e a terceira informação, podemos colocar um incremento ou decremento, para que o start chegue a condição de parada. Dentro das chavez iremos colocar o código que será repetido.

Incrementar ou Decremetar além de 1

Como a ultima informação é uma incrementação ou decrementação para que possamos chegar a condição de parada, normalmente colocamos ++1 ou --1, mas isso é um problema se queremos que pular mais que um por um. Quando incrementamos o número ou decrementamos, não estamos fazendo nada mais que i = i +1, ou sinal de menos para decrementar. Outra maneira de expressar isso é com i+=1, então podemos simplesmente substituir o 1 e colocar o valor ou sinal desejado.

Exemplo: Percorrer um array

Para que possamos fazer um for percorrer um array, temos que tomar alguns cuidados, como por exemplo, a segunda informação, normalmente colocamos um número fixo para condição de parada, mas dessa vez demos que colocar tamanho do array. Podemos até colocar um núemro fixo, mas se essa array crescer, o for não será mais tão útil, já que não pegará mais todos os valors. Para que possamos contornar, podemos colocar um .length na condição de parada, resolvendo a situação, onde ele tem a capacidade de analisar quandos valores o array possue, mas temos novamente um problema, queremos percorrer o array, e não descobrir quantos elementos ela tem. a Indexação do array começa com 0, enquando o .lenght começa por 1, então temos que subtrair -1 para que possa funcionar perfeitamente.
