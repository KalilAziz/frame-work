Operadores de comparação

Os operadores de comparação servem para comparar dois valores, sendo eles: < , <=, >, >= , ==, ===, != , !===

                                                                                                                                                                                                                                                                                                                 

Operadores de comparação: >

> : Este sinal significa maior que, lembrando que os sinais de comparação sempre retorna um valor booleano, sendo true ou false. No caso do sinal maior que, se o valor1 for maior que o valor2, será true, caso contrário, será false.

Operadores de comparação: ==

== : Este sinal significa igualdade, que pode ser confundido com o sinal de atribuição (=), lembrando que os sinais de comparação sempre retorna um valor booleano, sendo true ou false. O sinal de igualdade checa a igualdade entre dois operandos, tanto da esquerda quando da direita, checando apenas os valores, e não os tipos. Então um number 7 e string '7' são iguais. Por conta dessa caracterísitica, não é recomendado sua utilziação.

Operadores de comparação: ===

=== : Este sinal significa igualdade restrita, que pode ser confundido com o sinal de atribuição (=), lembrando que os sinais de comparação sempre retorna um valor booleano, sendo true ou false. O sinal de igualdade restrita checa os operandos, tanto da esquerda quando da direita, checando os valores e se os tipo são iguais. Então um number 7 e string '7' são diferentes, por que apesar de possuirem o mesmo valor, um é do tipo Number e o outro é do tipo String.
