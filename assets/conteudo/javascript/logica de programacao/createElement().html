createElement

Utilizando o createElement temos a possibilidade de criar elementos html e inserir pelo js. Para que conseguirmos uitlizar o createElement, precisamos apenas colocar o nome do lugar que iremos criar esse elemento, normalmente utilziado em conjunto com o document, posteriormente usamos a marcação de pontos e createElement(). Dentro dos parenteses, colocamos entre aspas o elemento que queremos criar.
