Style

Podemos usar o style para que possamos alterar um estilo css em elementos selecionados. Para que possamos usar o style, temos que colocar o elemento que desejamos fazer alterações, posteriormente temos que colocar marcação de ponto, style e por ultimo colocamos a estilização css. Mas lembrando que a estilização precisa ser em CamelCase, ou seja, não pode ter espaços no nome, e se tiver uma segunda palavra, a primeira palavra começa com letra minuiscula e a segunda palavra com letra maiúscula. Então, background color vira backgroundColor.

obs: A maneira mais comum de selecionar um elemento é atravéz do querySelector.
