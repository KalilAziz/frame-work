Object dateD

Com a utilização do object Date, podemos trabnalhar com datas. Por ele ser um objeto construtor, precisamos utilizar a palavra new na sua construção. Uma caracterísitica é que ela sempre começa com a letra maúscula. O mais comumente usado é sendo atribuido a uma variável. Para que possamos usar o object Date, temos que declarar uma variável, posteriormente atribuimos a o object date, que teremos que colocar new Date(), onde colocamos as datas desejadas

Para que possamos visualizar a data, temos que colocar o nome da variável que atribuimos o Object Date e .toString()
