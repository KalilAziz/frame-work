setAttribute

Poderemos usar o setAttribute para que possamos setar um atributo em algum elemento html, podendo escutar o nome do atributo e o valor que iremos definir para ele. Para que possamos usar o setAttribute(), iremos passar dois valores dentro dos parenteses, que estarão entre parenteses e separados por vírgula. O primeiro valor é o nome do atributo e o segundo valor será o valor que esse atributo terá.
