Atribuições via desestruturações - Array

A atribuioção via desestruturação é uma nova forma que poderemos fazer atribuições de estruturas, podemos fazer essas atribuições de várias variáveis simuntâneamente. Com a atribuição via desestruturação, podemos "arrancar" coisas do array e já utilizar nessa sintax. Lembrando que se inicializar com const, todas as astribuições seguintes terão que ser com const, e com let a mesma coisa. Para que possamos utilizar atribuição via desestrituração com arrays, temos que declara uma constrante com clochetes e atribuimos a array que queremos retirar os valores. Dentro dos colchetes, colocamos o nomes, que se tornarão variáveis com os valores retirados do array. O primeiro nome será o primeiro valor do array, o segundo nome será o segundo valor do array e por ai vai.

Exemplo:

const array = [100, 200, 300, 400, 500, 600, 700]

const [primeiroValor, segundoValor, terceiroValor] = array
