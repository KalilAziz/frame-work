Objetos

O objeto serve para agrupar muitas variáveis em um lugar só, sendo mais simples do que ficar criando muitas variáveis e sair enumerando elas. A forma de criação de um objeto é muito semelhante com a forma de criação de um array, mas se difere que em vez de utilziar colchetes, utilizamos chavez. E dentro das chavez podemos criar atributos, muito parecido com variáveis, só que está dentro de um objeto, não sendo necessário a utilização da palavra const nem let.

Criação de um objeto

A criação de um objeto em si é muito semelhante a criação de um array, onde temos que iniciar uma variável com um nome e fazer uma atribuição com o sinal de igual, mas a diferença é que o array atribuimos colchetes, enquanto no array atribuimos chaves. Dentro das chaves colocamos os atributos, que são uma espécie de variáveis que carrega valores, sendo necessário colocar apenas o nome, e não let ou const. Para atribuir valores a esses atributos, basta colocar : e o que desejamos.

Acesso de atributo de um objeto com funtion

O problema da utilização de objetos dessa forma é que caimos na mesma pergunta que fizemos anteriormente em funções, não seria mais fácil utilizar variáveis normais que dariam menos trabalho? seria, mas podemos evoluir nosso Objeto também, colocando-o para trabalhar em conjunto com uma função, unindo o útil ao agradável.

Devemos lembrar que a criação de um objeto depende principalmente das chaves que são seu corpo, e dos atributos que carregam os valores, não precisa estar necessariamente vinculado ou sendo atribuido a uma variável, então podemos criar uma função e atribuir um objeto a seu return, e quando a função for executada, o objeto será executado junto.

Outra alteração que temos que fazer é que, em vêz de setar os valores dos atributos, passamos os parâmetros das funções, pois quando chamarmos a função com seus argumentos, os objetos receberão seus calores automaticamente e serão atribuidos aos atributos.

Para chamar os objetos é bem simples, temos duas formas, a primeira delas é chamar os objetos com a notação de ponto quando formos chamar afunção, mas ficaria bem verboso e complicado, então a melhor forma seria atribuir a chamada da função com seus argumentos a uma variável, e posteriormente, utilizar o nome da variável para fazer notação de ponto.

Modernizando o cesso de atributo de um objeto com Function

A modernização do objeto na utilização com a função é uma simples modificação, onde se os valores dos parametros das funções forem iguais aos nomes dos atributos, não precisamos declarar os paramentros para atribuir os valores aos atributos, precisando apenas colocar o nome dos parametros iguais aos atributos que já estarão naturalmente conectados, mas lembre-se que os nomes devem estar iguais.

Function dentro de um objeto

Da mesma forma que temos como usar os objetos em um return de uma função, podemos usar funções como um dos atributos de um objeto. Quando usamos a função como um dos atributos, podemos chama-la de método do objeto.

Para que possamos declarar o método, não precisamos declarar o function, sendo necessário apenas o nome da função, os parenteses dos parâmetros e as chaves do corpo da função.

Vale lembrar que se quisermos usar os atributos dos objetos no método, temos que usar a palavra this com notação de ponto, que permitirá referenciar ao proprio elemento que está chamando o método e usar seus valores internos.

this com notação de ponto - this.atributo

Para chamar o método, é muito parecido com a chamada do objeto, temos que colocar o nome do objeto, uma notação de ponto e o nome da função, acompanhada de seus parenteses.
