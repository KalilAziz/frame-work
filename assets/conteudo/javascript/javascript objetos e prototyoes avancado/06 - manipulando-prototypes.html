Manipulando prototypes

Podemos colocar um objeto como prototype de outro, onde o objeto principal terá acesso a todos os métodos e tributos do objeto que foi inserido, e para que possamos fazer isso, temos que usar o Object.setPrototypeOf(). Para que possamos utiliza-lo, temos que declarar Object.setPrototypeOf(), onde dentro dos parenteses, iremos colocar duas informações separados por vírgula, a primeira é o objeto principal e a segunda é o objeto que iremos setar como prototype do objeto principal.

See the Pen 094 - Manipulando Prototypes by KalilAziz (@kalilaziz) on CodePen.

Setando novos atributos em constructor function por prototype

Para que possamos setar novos atributos em uma constructor function, precisamos utilizar o Object.setPrototypeOf, mas a diferença é que os novos atributos estarão dentro de um objeto declarado normalmente, e quando for passar a ordem de quem será o prototype de quem, iremos colocar a constructor function como prototype do objeto.
