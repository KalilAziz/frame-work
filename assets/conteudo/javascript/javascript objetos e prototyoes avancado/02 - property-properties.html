Object.defineProperty e Object.defineProperties

Como visto anteriormente, podemos usar o freeze para congelar todas as propriedades do objeto, mas se quisermos ter um controle mais preciso, é impossível de conseguirmos utilizando o freese, e por isso que podemos usar o Object.defineProperty() para poder fazer a manipulação do estado de um atributo ou Object.defineProperties() para vários atributos.

Object.defineProperty

Como dito anteriormente, podemos usar o defineProperty para que possamos travar apenas um do atributos de um objeto. Para que possamos utilizá-lo, iremos primeiro criar nossa constructor function com os atributos desejados, para aqueles atributos que queremos manipular seu estado, não iremos declará-lo, mas sim declarar Object.defineProperty(), onde iremos passar três informações separados por vírgulas, como estamos usando constructor function, sempre colocamos this antes dos atributos, então a primeira informação será o this, a segunda informação será o nome do nosso atributo entre aspas, e por ultimo, a terceira informação será abre e fecha chavez. Dentro das chavez iremos informar quatro informações, que são: enumerable (Iremos colocar true ou false se queremos que a chave seja exibida), value ( iremos colocar o nome do parâmetro da constructor function que desejamos atribuir ao atributo), writable ( colocaremos true ou false se queremos que o atributo seja editável), configurable (colocaremos true ou false se podemos apagar a chave).

Object.defineProperties

Como dito anteriormente, podemos usar o defineProperties para que possamos travar vários atributos de um objeto. Para que possamos utilizá-lo, iremos primeiro criar nossa constructor function com os atributos desejados, para aqueles atributos que queremos manipular seu estado, não iremos declará-lo, mas sim declarar Object.defineProperties(), onde iremos passar duas informações separados por vírgulas, como estamos usando constructor function, sempre colocamos this antes dos atributos, então a primeira informação será o this, a segunda informação será abre e fecha chaves. Dentro das chaves iremos colocar o nome do atributo, dois pontos (:) e abre e fecha chaves. Dentro dessas chaves, iremos informar quatro informações, que são: enumerable (Iremos colocar true ou false se queremos que a chave seja exibida), value ( iremos colocar o nome do parâmetro da constructor function que desejamos atribuir ao atributo), writable ( colocaremos true ou false se queremos que o atributo seja editável), configurable (colocaremos true ou false se podemos apagar a chave). A cada atributo, iremos colocar o seu nome, dois pontos e as chaves com as quatro informações dentro, para adicionar mais atributos, basta colcoar vírgula e refazendo o mesmo processo.
