Freeze

Podemos usar o freeze para que possamos congelar nosso array e impedir que ele sofra modificações, seja inserção, edição ou remoção de valores nos atributos do objetos. Para que possamos utilziar o freeze, temos que declarar Object.freeze(), onde colocaremos o dentro dos parenteses o nome do objeto que desejamos congelar.
