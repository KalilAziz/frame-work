...(spread)

Objetos trabalha por referência, ou seja, se atribuido a outra variável, se alterar a variável ou o objeto original, os dois se alterarão, pois apontam para o mesmo local da memória, para que essa ligação seja encerrada e termos objetos indepedentes, utilizamos o spred operator (...). Com spred operator, podemos também concatenar Objetos inteiros com atributos separados e criar um novo objeto com os atributos de todos os objeto concatenado. Para que possamos concatenar o objeto e atributos em um novo objeto, precisamos declarar uma variável, onde iremos atribuir abre e fecha chaves, e colocar spred operator juntamente com o nome do nome do objeto que iremos concatenar, e para colocar atributos, teremos que colocar o atributo como declararíamos dentro de um objeto.
