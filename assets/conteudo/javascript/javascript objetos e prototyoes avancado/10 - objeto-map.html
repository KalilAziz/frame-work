Objeto Map

O objeto map é realmente uma estrutura de dadeos, diferentemente do map que vimos em array. O map é utilziado em situações muito específicas, como nesse exemplo, onde iremos se tentamos objeter o resultado desejado pela forma convencional, ele irá alterar a ordem. E para que que possamos corrigir esse problema, precisamos utilizar o map. Para que possamos utiliza-lo, iremos utilizar o .set para inserir os dados, e .get, para visualizar os dados.
