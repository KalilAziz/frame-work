Getters e Setters

Para que possamos fazer a utilização do getter e setter com Object.defineProperty(), precisamos primeiro declarar uma constructor function normalmente, declarando os atributos e colocando os atributos que desejamos manupular os estados utilizando normalmente o Object.defineProperty(), mas a diferença é que além das quatro informações que colocamos normalmente ( enumerable, value, writable, configurable), colocaremos get e set. Lembrando que não é necessário colocar o value, já que vamos setar o valor usando o get, não é necessário colocar o writable se vamos usar o set, já que o set irá editar o valor. O get tem o papel de setar o valor do parâmetro da constructor function no atributo desejado. E para que possamos usar o get, temos que colocar get, dois pontos (:), e uma função anônima, onde colocamos return e o parâmetro da constructor function. Já o get, tem o papel de setar novos valores no nosso atributo. Para que possamos usar o set, temos que colocar set, dois pontos (:), e uma function anônima, onde iremos colocar um parâmetro. Posteriormente no final da função, iremos colocar o nome do nosso parâmetro da constructor function, e iremos atribuir o parâmetro da function anônima do set.
