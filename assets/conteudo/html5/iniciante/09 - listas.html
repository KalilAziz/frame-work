Listas

Para a criação de listas, existem duas possibilidades, as ordenadas e as não ordenadas.

Para que possamos criar a lista ordenada, primeiramente temos que colocar a tag <ol> </ol>, e dentro dela, as tags <li> </li>, onde serão colocados os itens das listas.

Já para a criação das listas não ordenadas, é muito semelhante a criação da lista ordenada, mas a diferença é que em vez de <ol> </ol>, é na verdade <ul> </ul>
