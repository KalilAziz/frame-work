Formatações de Textos

Negrito: Para que possamos marcar alguma parte do texto em negrito, semanticamente, é necessário utilizar a tag <strong>

Itálico: Para que possamos marcar alguma parte do texto em itálico, semanticamente, é necessário utilizar a tag <em>

Mark: Para que possamos marcar alguma parte do texto em marcação, semanticamente, é necessário utilizar a tag <mark>

small: Para que possamos marcar alguma parte do texto em letras menores, semanticamente, é necessário utilizar a tag <small>

del: Para que possamos riscar alguma parte do texto, semanticamente, é necessário utilizar a tag <del>

ins: Para que possamos marcar alguma parte sublinhada do texto, semanticamente, é necessário utilizar a tag <ins>

sub: Para que possamos marcar alguma parte do texto para baixo, semanticamente, é necessário utilizar a tag <sub>

sup: Para que possamos marcar alguma parte do texto para cima, semanticamente, é necessário utilizar a tag <sup>
