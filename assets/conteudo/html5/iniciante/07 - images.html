Images

Para podermos adicionar uma imagem, primeiramente precisamos colocar a tag <img src="" alt="">, onde src="" colocamos o diretório da imagem, seja na web ou no computador, e alt="" colocamos uma mensagem escrita para o usuário caso a imagem não carregue.
