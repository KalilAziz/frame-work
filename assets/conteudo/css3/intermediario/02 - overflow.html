Overflow

O overflow é quando temos um conteúdo dentro do elemento, só que esse conteúdo é maior que o elemento. O atributo overflow tem alguns valores, sendo eles:

visible: Permite o conteúdo aparecer, sendo o padrão;

hidden: Esconde o conteúdo que passa do elemento;

scroll: Desenvolve uma barra de rolagem para que possamos ver todo o conteúdo, mas o problema é que coloca barra de rolagem tanto na horizontal, quanto na vertical, mesmo que não precise;

auto: Coloca uma barra de rolagem apenas onde precisa;

overflow-y: scroll;: Coloca uma barra de rolagem no eixo na vertical;

overflow-x: scroll;: Coloca uma barra de rolagem no eixo na horizontal e vertical (se necessário)

OBS: Funciona com qualquer tipo de conteúdo, inclusive com imagens
