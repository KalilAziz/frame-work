Estilos de links

Quando criado um link com a tag <a href=""> </a >, ele de padrão vem pré definido com a cor azul e com um traço embaixo dele.

Para que possamos alterar esse traço, podemos definir um text-decoration: none;.

Para trocar a cor do link, bastamos utilizar uma propriedade color:;

Podemos mudar a cor de fundo com background-color:;.

Para dar mais espaço para o link, podemos colocar padding:;.

podemos colocar border-radius:; para arredondar as pontas.

Podemos definir também bordas, com border:;.

Podemos utilizar um :hover, para que quando o mouse tiver em cima, ele terá outras configurações.
