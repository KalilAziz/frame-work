Classes e Id

Os Id e Classes servem para identificar uma tag ou elemento, que na maioria das vezes é utilizado para aplicar estilos a esses tipos, a diferença deles é que o Id serve como identificação de apenas uma tag ou elemento, enquanto a classe pode ser usado para vários ao mesmo tempo, onde pode atribuir o nome dessa classe a vários outros elementos ao mesmo tempo. Eles não se diferem apenas no conceito, mas na declaração dos elementos e no css.

Para declarar a Classe em um elemento, utilizamos class="".

Para declarar a Classe no css, utilizamos um "." e o nome da classe.

Para declarar o Id em um elemento, utilizamos id="".

Para declarar o Id no css, utilizamos um "#" e o nome da classe.
